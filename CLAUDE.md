# Voice Research

## Setup

- Package manager: `uv`
- Python: 3.12
- Virtual env: `.venv` (local to this directory)

## Common commands

- Install a package: `uv add <package>`
- Remove a package: `uv remove <package>`
- Run a script: `uv run python <script.py>`
- Activate venv manually: `source .venv/bin/activate`
